GoBildaPinpointDriver

@I2cDeviceType
@DeviceProperties(name = "goBILDA速 Pinpoint Odometry Computer", xmlTag = "goBILDAPinpoint", description = "goBILDA速 Pinpoint Odometry Computer (IMU Sensor Fusion for 2 Wheel Odometry)")
open class GoBildaPinpointDriver : I2cDeviceSynchDevice<DEVICE_CLIENT>

The goBILDA速 Pinpoint Odometry Computer driver, ported from goBILDA速.

To use this in BunyipsLib, use this as any other HardwareDevice (such as DcMotor).

File copy acquired 25th March 2025, keeping the public API surface backwards compatible with other distributions of the driver.
