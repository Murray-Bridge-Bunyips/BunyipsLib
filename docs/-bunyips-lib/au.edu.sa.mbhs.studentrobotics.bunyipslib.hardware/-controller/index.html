Controller

open class Controller : Gamepad

A wrapper around a Gamepad object that provides a Controls interface and custom input calculations. These gamepad objects are used natively in BunyipsOpMode, and are the drop-in replacements for gamepad1 and gamepad2. Controller instances are injected into the FtcEventLoop through inject.
