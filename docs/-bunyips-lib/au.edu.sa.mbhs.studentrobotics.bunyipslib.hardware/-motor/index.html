Motor

open class Motor : SimpleRotator, DcMotorEx

Drop-in replacement for a DcMotor that uses custom control algorithms to operate RUN_USING_ENCODER and RUN_TO_POSITION modes. Internally integrates a gain scheduler to allow for more precise system coefficients against gravity and other external forces. This class effectively wraps an entire DcMotor and regulates all the operations.

This class is designed to be downcasted to a DcMotor where used, integrating with the current motor modes while providing faster, predictable control systems.
