open fun enable(processor: AprilTag, @NonNull localizer: Localizer): AprilTagPoseEstimator

Create a new AprilTagPoseEstimator runner.

Note that the option to also update heading based off these readings is enabled by default. Disable with setHeadingEstimate.

Also note that by default, this pose estimator will assume the camera is exactly at the center of the robot, facing forward. Adjust this using setCameraOffset.

a new AprilTagPoseEstimator instance
