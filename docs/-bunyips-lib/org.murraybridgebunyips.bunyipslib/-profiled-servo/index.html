ProfiledServo

open class ProfiledServo : ServoImpl, PwmControl

Extension of the extended Servo interface that allows for motion profiling via a TrapezoidProfile. This extension also offers refresh rate and cache tolerance handling for loop time optimisation.

This class serves as a drop-in replacement for the Servo, similar to Motor with the DcMotor. Do note that this class cannot be casted to a ServoImplEx instance, but it does implement the extended PwmControl interface for extended operations.
