DeadlineTaskGroup

open class DeadlineTaskGroup : TaskGroup

A group of tasks that runs all at once, until the first queued task is finished. Once this task is finished, all other tasks are cancelled if not completed. The task that will be the decider for the deadline is the first task passed as an argument.
