Cannon

open class Cannon : BunyipsSubsystem

Class for a generic servo switch that may switch between two known states. A whole new file is technically unnecessary, but we wanted to make a class named Cannon.

"Fire in the hole!"

This class is deprecated by Switch, which offers more functionality relating to single-servo subsystems, allowing movement beyond just two positions.
