CartesianMecanumDrive

open class CartesianMecanumDrive : BunyipsSubsystem

Base class for a Cartesian drive system that uses Mecanum wheels. Includes all the common math done across all Mecanum drive systems. This system is deprecated by RoadRunner, but remains for legacy/non-RoadRunner purposes.
