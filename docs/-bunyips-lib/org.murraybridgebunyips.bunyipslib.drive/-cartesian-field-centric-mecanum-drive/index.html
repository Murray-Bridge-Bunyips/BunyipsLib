CartesianFieldCentricMecanumDrive

open class CartesianFieldCentricMecanumDrive : CartesianMecanumDrive

Variant of the Cartesian MecanumDrive that uses field-centric controls, accounting for robot heading. This system is deprecated by RoadRunner, but remains for legacy/non-RoadRunner purposes.
