IMUOp

open class IMUOp : BunyipsSubsystem

IMU utility class that will wrap an Inertial Measurement Unit to provide data and automatic updates of the IMU angles as part of a subsystem. This subsystem has no meaningful Task to schedule.

This class, alike the old IMUOp also supports different readings of IMU measurement, such as an unrestricted domain on heading reads, while also being able to provide IMU units in terms of WPIUnits.

Angles from this class are intrinsic. Read more in the YawPitchRollAngles and Orientation classes.
